# DevTools

开发者在线工具箱 — 纯前端实现，即开即用，数据不上传。

## 工具列表

| 工具 | 说明 |
|------|------|
| JSON 格式化 | 美化、压缩、转义、语法高亮 |
| 时间戳转换 | 时间戳与日期互转、时区选择、批量转换 |

## 技术栈

Vue 3 + TypeScript + Vite + Tailwind CSS + shadcn-vue

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```
