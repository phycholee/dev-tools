## 一、产品概述

### 1.1 产品定位

DevTools 是一个面向开发者的在线工具集合网站，提供 JSON 处理、时间戳转换等常用开发工具。所有数据处理在浏览器端完成，无需后端服务，确保数据安全。

### 1.2 目标用户

- 程序员、前端/后端开发工程师
- 运维工程师
- 数据分析师
- 其他需要处理 JSON 和时间戳的技术人员

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 纯前端 | 数据不上传，隐私安全 |
| 快速 | 即开即用，无需注册 |
| 专业 | 面向开发者，功能实用 |
| 可扩展 | 工具注册机制，方便添加新工具 |

### 2.3 工具注册机制

```typescript
// src/tools/registry.ts
interface ToolDefinition {
  name: string           // 工具名称
  path: string           // 路由路径
  icon: string           // 图标
  description: string    // 简介
  category: string       // 分类（用于主页分组显示）
  component: () => Promise<Component>  // 懒加载组件
}

export const tools: ToolDefinition[] = [
  {
    name: 'JSON格式化',
    path: '/json-formatter',
    icon: '{ }',
    description: 'JSON美化、转义',
    category: '数据处理',
    component: () => import('./json-formatter/JsonFormatter.vue')
  },
  {
    name: '时间戳转换',
    path: '/timestamp-converter',
    icon: '⏱',
    description: '时间戳与日期互转',
    category: '时间日期',
    component: () => import('./timestamp-converter/TimestampConverter.vue')
  }
]
```

### 2.4 设计原则

1. **工具独立**：每个工具是一个独立 Vue 组件，有自己的路由和逻辑
2. **工具注册**：通过统一的工具注册机制，方便扩展
3. **响应式**：移动端适配，输入输出区域可堆叠
4. **无依赖**：核心功能不依赖第三方 API，纯本地计算

### 3.2 JSON 格式化工具

#### 功能列表

| 功能 | 说明 |
|------|------|
| 结构格式化 | 美化 JSON，支持自定义缩进（2/4 空格） |
| 压缩 | 去除所有空白，单行输出 |
| 转义 | 将 JSON 转为字符串转义格式（用于代码中） |
| 消除转义 | 将转义的 JSON 字符串还原 |
| 语法校验 | 检测 JSON 语法错误，显示错误位置 |
| 语法高亮 | 输出区域 JSON 语法着色 |

#### UI 布局

```
┌─────────────────────────────────────────────┐
│  JSON格式化工具                               │
├─────────────────────────────────────────────┤
│  ┌─────────────────┐  ┌─────────────────┐   │
│  │                 │  │                 │   │
│  │   输入区域       │  │   输出区域       │   │
│  │   (可编辑)      │  │   (语法高亮)    │   │
│  │                 │  │                 │   │
│  │                 │  │                 │   │
│  └─────────────────┘  └─────────────────┘   │
│                                             │
│  [格式化] [压缩] [转义] [消除转义] [复制] [清空]│
│                                             │
│  缩进: ○2空格 ○4空格  □排序key               │
└─────────────────────────────────────────────┘
```

#### 交互流程

1. 用户在左侧输入区域粘贴/输入 JSON
2. 点击功能按钮（格式化/压缩/转义等）
3. 处理结果在右侧输出区域显示，带语法高亮
4. 如有语法错误，在输入区域下方显示错误信息和位置
5. 点击复制按钮将结果复制到剪贴板

### 3.3 时间戳转换工具

#### 功能列表

| 功能 | 说明 |
|------|------|
| 时间戳→日期 | 支持秒/毫秒时间戳转可读日期 |
| 日期→时间戳 | 日期选择器转时间戳 |
| 当前时间戳 | 实时显示当前时间戳（秒/毫秒） |
| 时区转换 | 支持常见时区转换 |
| 批量转换 | 支持多行时间戳批量转换 |

```
┌─────────────────────────────────────────────┐

│  └───────────────────────────────────────┘  │
│                                             │
│  ┌───────────────────────────────────────┐  │
│  │ 批量转换                               │  │
```

#### 交互流程

1. **当前时间戳**：页面加载后实时更新，每秒刷新
2. **时间戳→日期**：输入时间戳，自动识别秒/毫秒，点击转换显示日期
3. **日期→时间戳**：选择日期时间和时区，自动计算时间戳
4. **批量转换**：每行一个时间戳，批量转换后显示结果

## 六、部署方案

### 6.1 推荐平台

| 平台 | 优势 | 说明 |
|------|------|------|
| Vercel | 全球 CDN，自动部署 | 连接 Git 仓库后自动构建部署 |
| GitHub Pages | 免费，简单 | 适合个人项目 |
| Netlify | 功能丰富 | 支持表单、函数等 |

**推荐 Vercel**：免费额度充足，全球 CDN 加速，支持自动部署。

### 6.2 部署流程

1. 将代码推送到 GitHub 仓库
2. 在 Vercel 导入项目
3. 配置构建命令：`npm run build`
4. 配置输出目录：`dist`
5. 自动部署完成

### 7.1 后续工具建议

| 工具 | 说明 |
|------|------|
| Base64 编解码 | Base64 字符串编码/解码 |
| URL 编解码 | URL 编码/解码 |

| Diff 对比 | 文本差异对比 |
| 颜色转换 | HEX/RGB/HSL 颜色转换 |
| UUID 生成 | 在线生成 UUID |
| Hash 计算 | MD5/SHA1/SHA256 计算 |

### 7.2 扩展方式

通过工具注册机制，添加新工具只需：

1. 在 `src/tools/` 下创建新工具目录
2. 开发工具组件
3. 在 `registry.ts` 中注册工具（包含 category 字段）
4. 自动出现在侧边栏、主页卡片列表和路由中

**主页自动更新**：主页通过读取 `registry.ts` 中的工具列表，自动生成分类分组的卡片展示，无需手动维护主页内容。

## 九、总结

DevTools 开发者工具箱是一个面向开发者的在线工具集合，首期包含 JSON 格式化和时间戳转换两个核心工具。采用 Vite + Vue 3 + TypeScript 技术栈，纯前端实现，暗色主题，响应式设计。通过工具注册机制，方便后续扩展更多工具。
