> 最后更新：2026-03-28

## 📋 项目背景

DevTools 是一个面向开发者的在线工具箱，提供 JSON 格式化、时间戳转换、URL 编解码、Cron 表达式解析等实用工具。

### 目标用户
- **主要用户**：通用开发者（前端、后端、全栈）
- **使用场景**：日常开发中的快速工具需求，强调效率和易用性

### 设计目标
- **品牌个性**：简洁现代（类似 Linear、Notion）
- **主题方向**：跟随系统偏好（自动检测，同时优化亮色/深色主题）
- **品牌识别度**：中等（有独特颜色和字体，但不夸张）

### 阶段 2：字体系统更新 (`/typeset`)

- [ ] 建立清晰的字体层次

**成功标准**：
- 字体组合有独特性，但不夸张

## 📝 决策记录

### 2026-03-28：设计方向确定
- **决策**：从深色优先转为跟随系统偏好
- **原因**：用户反馈想要更现代的亮色主题，同时保持深色模式选项
- **影响**：需要同时优化两个主题，工作量增加但体验更好

### 2026-03-28：字体选择
- **决策**：替换 Inter 为更有特色的字体
- **原因**：Inter 是 AI 生成界面的典型选择，缺乏品牌识别度
- **候选**：Plus Jakarta Sans、DM Sans、Satoshi
- **影响**：需要测试中英文混排效果

### 2026-03-28：帮助系统程度
- **决策**：标准级别（每个工具有使用指南和快捷键列表）
- **原因**：用户明确选择"标准"，平衡了完整性和开发成本
- **影响**：需要为每个工具编写说明文案
